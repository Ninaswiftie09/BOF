abrir un cmd o en la misma terminal se puede hacer eso, es mas recomendable usar un cmd

primer paso comprobar si se tienen descargados el node.js y el docker usando estas lineas de codigo
que se escriben en el cmd 

node -v
docker --version

de no tenerlos descargarlos

despues entrar al proyecto con la linea de comando 
cd C:dirreccion de donde tienen el codigo // (importante no abrirlo desde la dirreccion del folder general llamado
bof, para iniciar el proyecto tienen que poner la direccion donde ya se ven todos los codigos del frontend
digamos si tengo la carpeta bof donde hay mas carpetas en este caso front y backend, se tienen que meter a la
carpeta de front y copiar esa direccion para que si funcione el iniciar el proyecto)

instalar lo importante
instalar el mnp y el chars.js 

npm install
npm install chart.js

luego solo se coloca 
npm run dev

si este no funciona se coloca 
npm run serve
